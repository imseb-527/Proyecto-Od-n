Álgebra

El álgebra es una rama fundamental de las matemáticas que utiliza letras para representar números desconocidos y estudiar sus relaciones mediante operaciones matemáticas.

1. Expresiones Algebraicas

Una expresión algebraica es una combinación de números, letras (variables) y operaciones matemáticas (suma, resta, multiplicación, etc.).

🔤 Elementos de una expresión:

  • Coeficiente: el número que multiplica a la variable. Ej: en 5x, el coeficiente es 5.
  • Variable: la letra que representa un número desconocido. Ej: x, y, z.
  • Término: cada parte separada por + o −. Ej: en 3x + 2y − 5, hay tres términos.
  • Grado: el exponente más alto de la variable. Ej: en x² + x, el grado es 2.

📌 Ejemplo de expresión:

3x² + 2x - 7

  • Términos: 3x², 2x, -7
  • Coeficientes: 3, 2 y -7 (constante)
  • Variables: x
  • Grado: 2 (porque x está elevado al cuadrado)

🧮 Simplificación de expresiones:

Consiste en reducir la expresión combinando términos semejantes.

Ejemplo: 5x + 3x - 2 = (5 + 3)x - 2 = 8x - 2

🧪 Evaluación de expresiones:

Consiste en sustituir la variable por un número y resolver.

Ejemplo: Si x = 2, entonces en la expresión 4x - 1:

  • Reemplazamos: 4(2) - 1 = 8 - 1 = 7

🧠 Otro ejemplo completo:

Expresión: 2x² + 3x - 5

Si x = -1:

  1. 2(-1)² + 3(-1) - 5
  2. 2(1) - 3 - 5 = 2 - 3 - 5 = -6

2. Ecuaciones de Primer Grado

Una ecuación de primer grado es una igualdad que contiene una incógnita (generalmente x) elevada a la potencia 1. Se llaman de "primer grado" porque la variable no está elevada a una potencia mayor que uno.

📘 Forma general:

ax + b = c, donde:

  • a es el coeficiente de la variable x
  • b es el término independiente
  • c es el resultado de la ecuación

🔍 ¿Cómo se resuelven?

El objetivo es despejar la incógnita (x). Para eso, se aplican operaciones inversas (suma, resta, multiplicación o división).

✅ Ejemplo 1:

Resolver: x + 5 = 12

  1. Restamos 5 en ambos lados: x + 5 - 5 = 12 - 5
  2. Resultado: x = 7

✅ Ejemplo 2:

Resolver: 3x - 4 = 11

  1. Sumamos 4 en ambos lados: 3x = 15
  2. Dividimos entre 3: x = 15 / 3
  3. Resultado: x = 5

📌 Verificación:

Para comprobar si la solución es correcta, sustituimos el valor de x en la ecuación original.

Ejemplo: En la ecuación 3x - 4 = 11, si x = 5 → 3(5) - 4 = 15 - 4 = 11 ✔️

⚠️ Errores comunes:

  • No aplicar la misma operación a ambos lados de la ecuación
  • Olvidar el signo negativo
  • No simplificar correctamente los términos

🎯 Ejercicio propuesto:

Resuelve: 2x + 6 = 18

Ver solución paso a paso
  1. Restar 6: 2x = 12
  2. Dividir entre 2: x = 6

3. Ecuaciones de Segundo Grado

Una ecuación de segundo grado es una igualdad que tiene la forma ax² + bx + c = 0, donde:

  • a, b y c son números reales
  • x es la variable
  • a ≠ 0 (porque si a = 0, no sería de segundo grado)

📘 Tipos de soluciones:

Las soluciones pueden ser reales o complejas, y se obtienen usando la fórmula general:

x = (-b ± √(b² - 4ac)) / 2a

El valor dentro de la raíz (b² - 4ac) se llama discriminante.

  • Si es positivo → 2 soluciones reales diferentes
  • Si es cero → 1 solución real (doble raíz)
  • Si es negativo → 2 soluciones complejas

✅ Ejemplo 1: Con 2 soluciones reales

Resolver: x² - 5x + 6 = 0

  1. Identificar: a = 1, b = -5, c = 6
  2. Aplicar fórmula general:
    x = (-(-5) ± √((-5)² - 4(1)(6))) / 2(1)
    x = (5 ± √(25 - 24)) / 2
    x = (5 ± √1) / 2
  3. Soluciones: x = (5 + 1)/2 = 3, x = (5 - 1)/2 = 2

✅ Ejemplo 2: Con una solución real (raíz doble)

Resolver: x² - 4x + 4 = 0

  1. a = 1, b = -4, c = 4
  2. Discriminante: (-4)² - 4(1)(4) = 16 - 16 = 0
  3. Fórmula: x = (4 ± 0) / 2 → x = 2
  4. Solución: x = 2 (raíz doble)

✅ Ejemplo 3: Con soluciones complejas

Resolver: x² + x + 1 = 0

  1. Discriminante: 1² - 4(1)(1) = 1 - 4 = -3
  2. Como el discriminante es negativo, las soluciones son complejas:
  3. x = (-1 ± √(-3)) / 2 → x = (-1 ± i√3) / 2

🎯 Ejercicio propuesto:

Resuelve: x² - 2x - 3 = 0

Ver solución paso a paso
  1. a = 1, b = -2, c = -3
  2. Discriminante: (-2)² - 4(1)(-3) = 4 + 12 = 16
  3. x = (2 ± √16) / 2 → x = (2 ± 4) / 2
  4. Soluciones: x = 3 y x = -1

4. Identidades Notables

Son fórmulas que simplifican productos especiales. Se aprenden por su utilidad en factorización y desarrollo de productos.

📌 Principales identidades:

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • (a + b)(a - b) = a² - b²

🔍 Ejemplo 1: Cuadrado de una suma

Expandir: (x + 3)²

  1. Aplicamos la fórmula: (a + b)² = a² + 2ab + b²
  2. a = x, b = 3
  3. Resultado: x² + 6x + 9

🔍 Ejemplo 2: Producto de una suma y una resta

Expandir: (x - 5)(x + 5)

  1. Usamos la fórmula: (a - b)(a + b) = a² - b²
  2. a = x, b = 5
  3. Resultado: x² - 25

🔍 Ejemplo 3: Cuadrado de una resta

Expandir: (2x - 4)²

  1. Fórmula: (a - b)² = a² - 2ab + b²
  2. a = 2x, b = 4
  3. Resultado: 4x² - 16x + 16

🎯 Ejercicio propuesto:

Expande: (3x + 2)²

Ver solución paso a paso
  1. Identificar: a = 3x, b = 2
  2. Aplicar fórmula: (a + b)² = a² + 2ab + b²
  3. (3x)² + 2(3x)(2) + (2)² = 9x² + 12x + 4
  4. Resultado final: 9x² + 12x + 4

5. Factorización

Es el proceso de escribir una expresión algebraica como el producto de dos o más factores. Es muy útil para resolver ecuaciones, simplificar expresiones y entender la estructura de los polinomios.

📚 Principales métodos de factorización:

  • 1. Factor común: Consiste en sacar el mayor factor común (número o variable) que se repite en todos los términos.
  • 2. Diferencia de cuadrados: Se aplica cuando tienes dos términos cuadrados perfectos separados por un signo menos: a² - b² = (a - b)(a + b).
  • 3. Trinomio cuadrado perfecto: Reconocer expresiones del tipo a² ± 2ab + b², que se factorizan como (a ± b)².
  • 4. Trinomio de la forma ax² + bx + c: Es el método para factorizar trinomios cuando a ≠ 1, usando el producto y la suma.

🔍 Ejemplo 1: Factor común

Expresión: 6x² + 12x

  1. Observamos que 6x es común en ambos términos.
  2. Dividimos cada término entre 6x: (6x² ÷ 6x = x), (12x ÷ 6x = 2)
  3. Resultado: 6x(x + 2)

🔍 Ejemplo 2: Diferencia de cuadrados

Expresión: x² - 16

  1. Reconocemos que x² es (x)² y 16 es (4)².
  2. Se aplica la fórmula: a² - b² = (a - b)(a + b)
  3. Resultado: (x - 4)(x + 4)

🔍 Ejemplo 3: Trinomio cuadrado perfecto

Expresión: x² + 6x + 9

  1. Identificamos: x² es (x)² y 9 es (3)²
  2. El término medio es 2·x·3 = 6x
  3. Se cumple la forma: a² + 2ab + b² = (a + b)²
  4. Resultado: (x + 3)²

🔍 Ejemplo 4: Trinomio de la forma ax² + bx + c

Expresión: 2x² + 5x + 3

  1. Multiplicamos a·c: 2·3 = 6
  2. Buscamos dos números que multiplicados den 6 y sumen 5 → 2 y 3
  3. Reescribimos el término del medio: 2x² + 2x + 3x + 3
  4. Agrupamos: (2x² + 2x) + (3x + 3)
  5. Factor común en cada grupo: 2x(x + 1) + 3(x + 1)
  6. Factorizamos por agrupación: (2x + 3)(x + 1)

🎯 Ejercicio propuesto:

Factoriza: 9x² - 25

Ver solución paso a paso
  1. 9x² = (3x)² y 25 = (5)²
  2. Es una diferencia de cuadrados: a² - b² = (a - b)(a + b)
  3. Resultado: (3x - 5)(3x + 5)

6. Inecuaciones

Las inecuaciones son desigualdades algebraicas que indican que una expresión es mayor, menor, mayor o igual, o menor o igual que otra. Se representan con los signos: <, >, ≤, ≥.

Se resuelven de manera similar a las ecuaciones, pero con una regla clave: al multiplicar o dividir por un número negativo, el signo de la desigualdad se invierte.

📌 Tipos de desigualdad:

  • x < a: x es menor que a
  • x > a: x es mayor que a
  • x ≤ a: x es menor o igual que a
  • x ≥ a: x es mayor o igual que a

🔍 Ejemplo 1: Inecuación simple

2x - 5 < 7

  1. Sumamos 5 a ambos lados: 2x < 12
  2. Dividimos entre 2: x < 6 ✅

⚠️ Regla importante:

Si multiplicas o divides ambos lados por un número negativo, debes invertir el signo.

✏️ Ejemplo 2: Multiplicación por número negativo

-3x > 9

  1. Dividimos entre -3: x < -3 (¡se invierte el signo!)
  2. Resultado: x < -3 ✅

🔍 Ejemplo 3: Inecuación con fracciones

(x - 2)/4 ≥ 3

  1. Multiplicamos ambos lados por 4: x - 2 ≥ 12
  2. Sumamos 2: x ≥ 14

🎯 Ejercicio propuesto:

Resuelve: 5 - 2x ≤ 11

Ver solución paso a paso
  1. Restamos 5 a ambos lados: -2x ≤ 6
  2. Dividimos entre -2 e invertimos el signo: x ≥ -3

📈 Representación gráfica (opcional):

Las soluciones se pueden representar en la recta numérica, usando puntos abiertos (◯) para < o >, y puntos cerrados (●) para ≤ o ≥, con una flecha indicando el rango.
