Álgebra

El álgebra es una rama de las matemáticas que estudia la combinación de elementos matemáticos utilizando letras, números y signos.

1. Expresiones Algebraicas

Son combinaciones de números, letras y signos que representan una operación matemática. Por ejemplo: 3x + 5 es una expresión algebraica.

Las letras representan variables y los números coeficientes.

2. Ecuaciones de Primer Grado

Son igualdades con una variable que está elevada a la potencia uno. Se resuelven aislando la incógnita. Ejemplo: x + 3 = 7.

Solución: x = 4.

3. Ecuaciones de Segundo Grado

Tienen la forma ax² + bx + c = 0 y se resuelven con la fórmula general o factorizando.

Fórmula general: x = (-b ± √(b² - 4ac)) / 2a.

4. Identidades Notables

Son fórmulas que permiten simplificar productos. Las más comunes son:

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • (a + b)(a - b) = a² - b²

5. Factorización

Es el proceso de escribir una expresión como producto de factores. Por ejemplo:

x² - 9 = (x + 3)(x - 3)

Se utilizan métodos como: factor común, trinomio cuadrado perfecto, diferencia de cuadrados, entre otros.

6. Inecuaciones

Son desigualdades algebraicas que se resuelven de forma similar a las ecuaciones, pero cuidando el sentido de la desigualdad.

Ejemplo: 2x - 5 < 7 → x < 6
